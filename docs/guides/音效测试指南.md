## 🔍 故障排除

### 听不到音效？
1. **检查设备音量** - 媒体音量，不是铃声音量
2. **确认游戏设置** - 音效开关是否开启
3. **重启应用** - 完全退出后重新打开
4. **检查文件** - 确认 `assets/sounds/` 目录中有音效文件

### 音效延迟？
1. **设备性能** - 老旧设备可能有延迟
2. **文件大小** - 较大的wav文件可能加载慢
3. **后台应用** - 关闭其他占用音频的应用

### 音效音质问题？
1. **生成的音效** - tap.wav, combo.wav, swap.wav 是简单生成的音调
2. **真实音效** - match.mp3 是真实下载的高质量音效
3. **可以替换** - 按照音效下载指南替换为更高质量的音效
